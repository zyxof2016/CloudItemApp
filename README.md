# 云朵识物乐园

一款专为2-6岁儿童设计的安卓原生启蒙识物应用。

## 项目简介

云朵识物乐园通过趣味互动的方式帮助儿童认识常见物品，培养观察力、记忆力和语言表达能力。应用采用现代化的MVVM架构，使用Kotlin和Jetpack Compose开发。

## 主要功能

### 1. 认物学习
- 分类展示物品图鉴（动物、水果、蔬菜、交通工具、日常用品等）
- 物品详情展示（图片、名称、读音、描述）
- 点击学习，左右滑动切换
- 自动记录学习进度

### 2. 趣味游戏
- **猜猜看游戏**：根据物品特征猜测物品名称
- **计分系统**：记录得分和正确率
- **难度分级**：适合不同年龄段的儿童
- **游戏记录**：保存游戏历史

### 3. 成就系统
- 学习成就（认识物品数量）
- 游戏成就（完成游戏、获得高分）
- 连续学习奖励
- 成就徽章展示

### 4. 个人中心
- 用户信息展示
- 学习统计
- 成就墙
- 收藏物品

### 5. 设置模块
- 音效开关
- 背景音乐开关
- 语言选择（中文/英文/双语）
- 字体大小调节
- 护眼模式

## 技术架构

### 技术栈
- **开发语言**：Kotlin
- **最低SDK版本**：Android 8.0 (API 26)
- **目标SDK版本**：Android 14 (API 34)
- **UI框架**：Jetpack Compose
- **架构模式**：MVVM + Clean Architecture
- **依赖注入**：Hilt
- **本地数据库**：Room
- **异步处理**：Kotlin Coroutines + Flow
- **图片加载**：Coil
- **动画**：Lottie

### 项目结构
```
app/
├── presentation/          # 展示层
│   ├── ui/               # UI组件
│   │   ├── main/        # 主界面
│   │   ├── learning/    # 学习界面
│   │   ├── game/        # 游戏界面
│   │   ├── profile/     # 个人中心
│   │   └── settings/    # 设置
│   ├── viewmodel/       # ViewModel
│   └── navigation/      # 导航
├── domain/              # 领域层
│   ├── model/          # 领域模型
│   ├── repository/     # 仓库接口
│   └── usecase/        # 用例
└── data/               # 数据层
    ├── local/          # 本地数据
    │   ├── database/   # 数据库
    │   ├── entity/     # 实体
    │   └── dao/        # 数据访问对象
    └── repository/     # 仓库实现
```

## 内容规划

### 物品分类
1. **动物世界**（40个）
2. **美味水果**（30个）
3. **新鲜蔬菜**（30个）
4. **交通工具**（35个）
5. **日常用品**（40个）
6. **自然现象**（20个）
7. **食物与饮料**（25个）
8. **身体部位**（15个）

### 难度分级
- **第一级（2-3岁）**：80个物品
- **第二级（3-4岁）**：80个物品
- **第三级（4-6岁）**：75个物品

## 构建说明

### 前置要求
- Android Studio Hedgehog (2023.1.1) 或更高版本
- JDK 17
- Android SDK 34
- Gradle 8.2

### 构建步骤

1. 克隆项目
```bash
git clone <repository-url>
cd CloudItemApp
```

2. 打开项目
```bash
# 使用Android Studio打开项目
```

3. 构建与运行
- **Debug构建**:
  ```bash
  ./gradlew assembleDebug
  ```
- **Release构建**:
  ```bash
  ./gradlew assembleRelease
  ```
- **安装并运行 (Debug)**:
  ```bash
  ./gradlew installDebug
  ```
- **运行单元测试**:
  ```bash
  ./gradlew test
  ```
- **清理项目**:
  ```bash
  ./gradlew clean
  ```
