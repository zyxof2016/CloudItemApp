### 2. 技术架构 ✅
- **架构模式**：MVVM + Clean Architecture
- **UI框架**：Jetpack Compose
- **依赖注入**：Hilt
- **本地数据库**：Room
- **异步处理**：Kotlin Coroutines + Flow
- **图片加载**：Coil
- **动画**：Lottie

## 技术亮点

1. **现代化架构**：采用Clean Architecture和MVVM模式，代码结构清晰，易于维护
2. **Jetpack Compose**：使用声明式UI，代码简洁，开发效率高
3. **依赖注入**：使用Hilt进行依赖管理，解耦性强
4. **响应式编程**：使用Flow进行数据流处理，实时更新UI
5. **护眼与提醒**：集成全局 EyeProtectionManager，符合儿童应用健康标准
6. **AI 赋能**：通过 PROMPTS.md 指导 AI 生成统一风格的视觉与音效资源

## 运行指南

### 前置要求
- Android Studio Hedgehog (2023.1.1) 或更高版本
- JDK 17
- Android SDK 34
- Gradle 8.2

### 构建步骤
1. 打开Android Studio
2. 打开项目：File -> Open -> 选择CloudItemApp目录
3. 等待Gradle同步完成
4. 点击运行按钮或使用命令：
   ```bash
   ./gradlew assembleDebug
   ./gradlew installDebug
   ```

## 注意事项

1. **图片资源**：当前代码已包含图片加载逻辑，但需将资源放入 `res/drawable` 目录。
2. **音频资源**：需按 `[ID]_cn.mp3` 命名规则放入 `res/raw`。
3. **动画资源**：loading.json 和 celebration.json 目前为占位符，需替换。
